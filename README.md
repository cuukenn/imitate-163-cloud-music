# h5

#### 项目介绍
名称：h5大作业</br>
基础：vue</br>
UI:vux</br>
数据流控制：vuex</br>
路由控制：vue-router</br>
字体图标来源：阿里矢量图标库</br>
目标：实现网易云大体界面和相关功能</br>
数据来源：考虑使用vue axios编写爬虫获取</br>
