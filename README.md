# h5

#### 项目介绍
名称：h5大作业\n
基础：vue\n
UI:vux\n
数据流控制：vuex\n
路由控制：vue-router\n
字体图标来源：阿里矢量图标库\n
目标：实现网易云大体界面和相关功能\n
数据来源：考虑使用vue axios编写爬虫获取\n
